# 📊 DecodeLabs Data Analytics – Project 3
### SQL Data Analysis

## 📌 Project Overview

This project is part of the DecodeLabs Industrial Training Program (Batch 2026). The objective is to perform SQL-based data analysis on an e-commerce sales dataset by writing queries that extract meaningful business insights.

The project focuses on applying fundamental SQL concepts such as filtering, sorting, grouping, and aggregation to answer real-world business questions.

---

## 🎯 Objectives

- Retrieve data using SQL `SELECT` statements
- Filter records using `WHERE`
- Sort results using `ORDER BY`
- Group records using `GROUP BY`
- Perform aggregations using:
  - COUNT()
  - SUM()
  - AVG()
  - MIN()
  - MAX()
- Generate business insights from transactional data

---

## 📂 Dataset

The dataset contains e-commerce order information including:

- Order ID
- Order Date
- Customer ID
- Product
- Quantity
- Unit Price
- Total Price
- Payment Method
- Order Status
- Coupon Code
- Referral Source
- Shipping Address
- Items in Cart

## ▶️ Running the Project

```bash

```

```bash

```

```bash
jupyter notebook
```

Run the SQL queries using SQLite, MySQL, PostgreSQL, or any compatible SQL database after importing the dataset.
